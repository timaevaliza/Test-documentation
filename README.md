# Тестирование веб-страницы, приложения 2ГИС.
- **Чек-лист:** для описания проверок функциональности веб-страницы с использованием разных видов тестирования.
- **Баг-репорт:** для документирования найденных дефектов
- Для открытия веб-страницы необходимо скачать архив test_quest_for_qa.zip, распаковать и открыть index.html
- Отдельным файлом приложены виды тестирования по цели, применимые для проверки мобильного приложения 2GIS
